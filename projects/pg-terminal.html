Overview

Web Terminal Interface
JS Experiment

A fully functional, browser-based terminal emulator built with vanilla JavaScript. It interprets commands, supports a simulated file system, and features classic retro CRT aesthetics.

Company Personal
My Role Developer

The Problem

01

Building a developer portfolio requires a unique way to present information that resonates with engineers.

02

Creating an interactive CLI in the browser without relying on heavy external libraries like xterm.js.

Process

Command Parser

Wrote a lightweight JavaScript parser to handle input strings, split arguments, and map them to executable functions.

Simulated File System

Created a JSON-based virtual file system allowing commands like `cd`, `ls`, and `cat` to traverse directories and read mock files.

Visual Aesthetics

Implemented CSS animations for the blinking cursor, text typing effects, and a subtle CRT scanline overlay using SVG filters.

The Solution

A highly interactive easter-egg for technical visitors.

Keyboard Handling

Full support for up/down arrow history cycling, tab-completion, and interrupt commands.

Extensible API

Easily extendable to add new commands by simply registering a new JS function to the command map.

Impact

A fun and engaging way to showcase JavaScript logic skills.

Vanilla Zero Dependencies
Easter Egg Highly Engaging

Technical Demonstration

Proves strong understanding of state management and string manipulation in JS.
